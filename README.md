# Choique.org website

## Requirements

1. [Install Zola](https://www.getzola.org/documentation/getting-started/installation/)
2. Clone the repository `https://github.com/emi420/choique.org`
3. Clone the theme `cd choique.org/themes && git clone https://github.com/semanticdata/mabuya`

## Getting started

```bash
zola serve
```
